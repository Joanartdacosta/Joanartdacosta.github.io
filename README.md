## Github Workflow Steps

- `git checkout -b "branch_name"`
- `git add .`
- `git commit -m "description"`
- `git push origin "branch_name"`
- Merge feature branch to main after PR is reviewed and approved
- `git checkout main`
- `git pull origin main`
